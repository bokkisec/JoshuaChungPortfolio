C++ Experience

I used C++ for most of my programming courses at Las Positas College (LPC) and UC Irvine (UCI).

UC Irvine

C++ was used in Data Structures (ICS 46).

Data Structures (ICS 46)

The Data Structures class at LPC counted as another class at UCI, so I took a similar course here. Implemented data structures and some algorithms. Although both classes were called Data Structures, I feel like I learned a lot of different things from both classes.

Projects involved:

  • Solving a Combinatorics Puzzle using recursion
  • Skip Lists
  • Hash Table using Cuckoo Hashing. Word Ladder using BFS
  • AVL Trees
  • Word Ladder again but using Priority Queue and Dijkstra's
  • Graph coloring to solve Register Allocation problem

Las Positas College

C++ was used in Computing Fundamentals I (CS 1), Computing Fundamentals II (CS 2), and Data Structures (CS 20).

Data Structures (CS 20)

This class required us to implement our own data structures and some algorithms. Also covered software design processes such as Waterfall and Agile.

Projects involved:

  • Array-Based List
  • Linked List
  • Stacks & Queues
  • Hash Table using Chaining
  • Heaps
  • Binary Search Tree Functions
  • Graph Navigation (DFS & BFS)

Computing Fundamentals II (CS 2)

This class introduced Object Oriented Programming and required us to work with raw pointers extensively.

Projects involved:

  • Bingo! Simulation
  • Implementing Linear & Binary Search, Bubble & Selection Sort
  • Hare vs Tortoise Race Simulation
  • Go Fish! Simulation
  • Calendar/To-Do List
  • Implementing our own version of std::vector
  • File I/O involving the use of an std::map

Computing Fundamentals I (CS 1)

This was the first college programming course I took. It taught the basics of computer programming from simple variables to creating structs. Projects in this course were simple programming tasks to help illustrate the concepts. While taking this course, I also watched about 80% of the 100 video C++ series created by The Cherno on Youtube as a supplement. I definitely learned more from the video series than the course.
